พื้นฐาน Git

Git เป็นเครื่องมือที่ช่วยติดตามการเปลี่ยนแปลงของไฟล์ในโปรเจกต์ โดยคำสั่งพื้นฐานที่ควรรู้มีดังนี้

git init

ใช้สร้าง Repository ใหม่ในโฟลเดอร์ปัจจุบัน

git add .

เลือกไฟล์ทั้งหมดที่ต้องการเตรียมเก็บไว้ใน stage

git commit -m "Add new feature"

บันทึกการเปลี่ยนแปลงเป็นเวอร์ชันใหม่

git status

ดูสถานะของไฟล์ว่าเปลี่ยนแปลงหรือยัง

  1. สร้างโฟลเดอร์และไฟล์โปรเจกต์
  2. รันคำสั่ง git init เพื่อเริ่มติดตาม
  3. ใช้ git add เพื่อเลือกไฟล์ที่ต้องการ
  4. ใช้ git commit เพื่อบันทึกเวอร์ชัน
