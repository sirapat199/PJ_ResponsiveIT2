การทำงานกับ GitHub

GitHub ช่วยให้คุณเก็บโค้ดบนเว็บและทำงานร่วมกับทีมได้อย่างสะดวก โดยกระบวนการทั่วไปมีดังนี้

  1. สร้าง Repository บน GitHub
  2. เชื่อม Repository กับเครื่องคอมพิวเตอร์ด้วยคำสั่ง git remote add origin ...
  3. ส่งโค้ดขึ้น GitHub ด้วยคำสั่ง git push -u origin main
  4. สร้าง Branch ใหม่เพื่อพัฒนา feature
  5. สร้าง Pull Request เพื่อขอ merge เข้า main
git clone <repository-url>

ดาวน์โหลดโครงการจาก GitHub มาใช้ในเครื่อง

git branch feature/login
git checkout feature/login

สร้างและสลับไปยัง branch ใหม่สำหรับการพัฒนา

git pull origin main

ดึงข้อมูลล่าสุดจาก branch หลักก่อนเริ่มงาน
