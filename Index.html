แนะนำการใช้ Git และ GitHub

เรียนรู้พื้นฐานการจัดการโค้ดด้วย Git และแชร์โปรเจกต์บน GitHub

Git คือเครื่องมือช่วยติดตามการเปลี่ยนแปลงของไฟล์ ส่วน GitHub คือแพลตฟอร์มสำหรับเก็บโค้ดและทำงานร่วมกับคนอื่นได้อย่างสะดวก

เริ่มเรียนรู้ Git ดู Workflow บน GitHub

สิ่งที่คุณจะได้เรียนรู้

  • เข้าใจความแตกต่างระหว่าง Git และ GitHub
  • ใช้คำสั่งพื้นฐานของ Git เช่น init, add, commit, push
  • สร้าง Repository และ Pull Request บน GitHub

Git คืออะไร

Git คือระบบควบคุมเวอร์ชันที่ช่วยติดตามและจัดการการเปลี่ยนแปลงของไฟล์ในโครงการ

GitHub คืออะไร

GitHub คือบริการเว็บที่ใช้ Git สำหรับเก็บโค้ด แชร์โปรเจกต์ และร่วมมือกันพัฒนาได้

เหตุผลที่ควรใช้

ช่วยเก็บประวัติการเปลี่ยนแปลง ทำงานร่วมกับทีม และป้องกันการสูญเสียไฟล์
